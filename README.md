 Number Guessing Game (Python)
A simple and interactive console-based game where the computer chooses a random number, and the user has to guess it.
This project is created as part of the VITyarthi Build Your Own Project requirement for the flipped course .
 Project Overview
The Number Guessing Game is a beginner-friendly Python application that helps understand basic programming concepts such as:
	•	Loops
	•	Conditional statements
	•	User input handling
	•	Random number generation
The game continues until the user guesses the correct number or chooses to reveal the answer.
Features
	•	Generates a random number between 1 and 100
	•	User can guess the number any number of times
	•	Special command “answer” to reveal the secret number
	•	Simple, clean, and beginner-friendly interface
	•	Immediate feedback after every guess
 Technologies Used
	•	Python 3
	•	Built-in random module
	•	Console/Terminal

Instructions for Playing
	1.	Run the Python file.
	2.	The program will pick a number between 1 and 100.
	3.	Enter your guesses.
	4.	If you want to know the answer, type answer.
	5.	If your guess matches the secret number → you win!
	6.	Otherwise → try again!
